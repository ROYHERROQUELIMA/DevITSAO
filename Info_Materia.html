Asignatura: Desarrollo Full Stack

Diseño, desarrollo e implementación de aplicaciones web modernas, accesibles y escalables.

Descripción

La asignatura de Desarrollo Full Stack forma al estudiante en la creación de aplicaciones web completas, integrando tecnologías de Front-End, Back-End, control de versiones y despliegue continuo.

Competencias

Front-End

HTML5, CSS3, diseño responsivo y accesibilidad.

Back-End

Arquitecturas escalables y patrones de diseño.

Herramientas

Git, Node.js, Visual Studio Code.

Frameworks

Angular, APIs REST y Firebase.

Temario

1. Fundamentos del Desarrollo Full Stack

Arquitectura cliente-servidor, roles del desarrollador y herramientas modernas.

2. Interfaces Web Modernas

HTML5 semántico, CSS3, Bootstrap/Tailwind y diseño responsive.

3. Patrones de Diseño

Singleton, Factory Method, Adapter, Decorator, Observer y State.

4. Desarrollo con Frameworks

Angular, consumo de APIs REST, formularios reactivos y Firebase.

Proyecto

Desarrollo de un sistema Full Stack aplicando análisis, planeación, ejecución y evaluación para resolver un problema real mediante tecnologías web modernas.
